Adarsh Yoga

Software Engineer at Intel Corporation

About

I am a compiler engineer in the Intel Python team. I work on designing programming abstractions for expressing GPU parallelism in Python. Before joining Intel, I went through the Santosh Nagarakatte Ph.D grind at Rutgers University. I worked on program analysis techniques for identifying performance and correctness issues in parallel programs.
